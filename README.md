# Sentiment Analysis of Tweets using Machine Learning

An AI-powered Sentiment Analysis System built using NLP, Machine Learning, Python, and Streamlit to classify tweets into Positive, Neutral, and Negative sentiments.

---

## Project Overview

This project analyzes social media tweets and predicts sentiment using Machine Learning and Natural Language Processing (NLP). The system is trained on the Sentiment140 dataset containing 1.6 million tweets and helps understand public opinion, customer feedback, and social media trends.

The project includes:

* Tweet preprocessing
* NLP text cleaning
* Feature engineering
* Machine learning model training
* Sentiment prediction
* Interactive Streamlit dashboard
